# FluidSort 💧

FluidSort is a mobile puzzle game where players transfer liquids between bottles in the correct order. Inspired by the real game *Magic Sort*, this version adds randomized level generation, color stacking mechanics, and advanced gameplay features.

## Features 🔧

- **Stack-Based Bottles:** Each bottle modeled as a Stack data structure to manage liquid layers using LIFO logic.  

- **Randomized Level Generation:** Levels distribute liquid colors randomly for varied gameplay each session.  

- **Stacked Pouring Mechanic:** When pouring, contiguous layers of the same color pour together for smoother merges.  

- **Modular OOP Scripts:** Scripts designed following object-oriented programming principles for maintainability and scalability.  

- **Platform:** Fully optimized for Android devices.  

- **Engine:** Developed with Unity for physics-based interactions and smooth gameplay.  

---

## How to Play 🎮

Drag and pour liquids between bottles to match colors in the correct order.  

- Stack same-colored liquids to pour multiple layers at once.  
- Complete levels by sorting all liquids correctly.  
- Randomized levels ensure each playthrough is different, increasing challenge and replayability.
