# FluidSort

A mobile puzzle game where players transfer liquids between bottles in the correct order.

## Features
- Modeled each bottle as a Stack data structure to manage liquid layers using LIFO logic
- Designed modular scripts following object-oriented programming principles
- Android platform
- Developed with Unity

## How to Play
Drag and pour liquids between bottles to match colors in the correct order. Complete levels by sorting all liquids correctly.

## Technologies
- Unity, C#
- Stack data structure
- OOP principles

This project is inspired by the real game: Magic Sort
